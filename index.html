Crunchyroll Auto Skip

Skip intros, credits, and auto-play next episode. 3 steps, ~30 seconds.

Step 1 — Install Tampermonkey

Tampermonkey is a free browser extension. It runs the auto-skip script for us. You only need to install it once.

Install Tampermonkey
Safari uses a different free app called Userscripts. After installing, open Safari → Settings → Extensions and enable it. Then click "I've installed it" below.

Step 2 — Install the Skip Script

Click the button below. Tampermonkey will pop up a dialog showing the script — click Install in that dialog.

Install Skip Script
Didn't see an install dialog?

If the script appeared as plain text in your browser, Tampermonkey isn't picking it up. Make sure step 1 is complete (the Tampermonkey icon should show in your browser toolbar), refresh this page, and try again.

Step 3 — Try it out

Open Crunchyroll and play any episode. Skip Intro, Skip Credits, and Next Episode buttons should now click themselves.

Open Crunchyroll
Verify it loaded: on Crunchyroll, open DevTools console (Cmd+Opt+J in Chrome / Cmd+Opt+C in Safari). You should see [cr-autoskip] active in orange.
Updates

Tampermonkey checks for script updates daily on its own. You'll get a small "update available" badge — click it once and you're current. No need to revisit this page.
